Checks if a given point on a given map is a frontier or not. Checks the occupancy grid values of the point ad its neighbours and determines whether or not it is a frontier.

Parameters
mapOccupancy grid of the surrounding. Unknown points correspond to -1, obstacles have a value greater than 0 and empty space will have the value 0. It is processed like a one dimensional array.
pointThe point in the array that needs to be checked as a frontier
mapSizeMap size is map height * map width
mapWidthWidth of the map
Returns
True if the given point is a frontier, False if not

Takes in the map and finds out the potential frontiers by doing a breadth first search. It uses a queue to maintain the frontiers. Makes calls to other methods of the class as a part of computing the frontiers.

Parameters
mapOccupancy grid of the surrounding. Unknown points correspond to -1, obstacles have a value greater than 0 and empty space will have the value 0. It is processed like a one dimensional array.
mapHeightVisually map is a two dimensional object but it is passed as a one dimensional array. So mapHeight is the height (or length) of the map.
mapWidthWidth of the map
posePose of the robot
Returns
Frontiers of type: a two dimensional integer vector
